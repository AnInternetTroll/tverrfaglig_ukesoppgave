# Development

Good old

You need to make a `users.db` file and set some enviormental variables (or in a `.env` file) before you can start

```sh
PORT=3000
DISCORD_ID=
DISCORD_SECRET=
CALLBACK_URL=
SQLITE=
SECRET=
```

```sh
npm i
npm start
```
